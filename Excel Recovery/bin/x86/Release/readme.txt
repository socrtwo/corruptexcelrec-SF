                              *****S2 Recovery Tools for Microsoft Excel 4.0.0*****

Note, some of the buttons won't work if there are invisible or visible Excel instances running in task manager. Before starting this program, hit Ctl-Alt-Delete start Task Manager and end all instances of Excel. Look for EXCEL.EXE or EXCEL.EXE * 32 (the latter will only appear if you have a 64 bit version of WIndows installed and a 32 bit version of Microsoft Office).

S2 Recovery Tools for Microsoft Excel is a program that tries to make easy, Microsoft's methods for recovering from Excel corruption. With the exception of the revert to last saved methods, it implements in one place all the suggestions for Excel recovery laid out by Microsoft at these locations: http://office.microsoft.com/en-us/excel-help/repairing-corrupted-files-in-excel-HA001034656.aspx and http://office.microsoft.com/en-us/excel-help/repairing-a-corrupted-workbook-HA010097017.aspx.

The program adds five non-MS methods for recovering data from corrupt workbooks that won't open. These are the "Non-MS Extract I", "Non-MS Extract II-XLSX Only", "Zip Repair-Try for XLS & XLSX", "Full Repair I XLSX Only" and "Full Repair II XLSX Only" methods. The first three use included freeware or open source executables from other projects to do their work, specifically the SILVERCODERS DocToText, No-Frills Unzipper, CMD Corrupt OfficeOpen2txt and Info-Zip projects. SILVERCODERS DocToText are available as open source on Sourceforge, http://sourceforge.net/, whereas the other two are available on my website: http://godskingsandheroes.info/software/.

There are earlier version of the program available if you contact me at socrtwo@s2services.com.

This program requires the installation of the .NET 2.0 framework. Hopefully most Windows machines have at least installed this version by now. Additionally, most of the functions require at least Excel 2003 to be installed on the machine. Please send me your reports of bugs to socrtwo@s2services.com.
